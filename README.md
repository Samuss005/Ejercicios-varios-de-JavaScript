# Ejercicios-varios-de-JavaScript
================================
Alumno: Samuele Isidoro Moccia
================================
Este repositorio contiene la resolución de 10 ejercicios básicos de JavaScript enfocados en la manipulación del DOM, eventos y lógica básica.

Cada ejercicio se encuentra en un archivo HTML independiente en la raíz del repositorio.
=============
Ejercicios:
=============
Ejercicio 1: Mostrar un mensaje al hacer clic

Archivo: ejercicio1.html
Crear un botón con id btn y un párrafo con id resultado. Al hacer clic en el botón, mostrar el texto: "¡Hola Clase!".

Ejercicio 2: Cambiar el color del texto

Archivo: ejercicio2.html
Un párrafo con id texto y un botón. Al pulsarlo, el texto debe cambiar a color rojo.

Ejercicio 3: Contador de clics

Archivo: ejercicio3.html
Crear un contador que aumente en 1 cada vez que se pulsa un botón y mostrar el resultado en pantalla.

Ejercicio 4: Mostrar la hora actual

Archivo: ejercicio4.html
Al hacer clic en un botón, mostrar la hora actual del sistema en un párrafo.

Ejercicio 5: Saludo personalizado con input

Archivo: ejercicio5.html
Un input para escribir el nombre, un botón y un párrafo. Mostrar: "¡Hola, [ nombre ]!".

Ejercicio 6: Calculadora de suma simple

Archivo: ejercicio6.html
Dos inputs numéricos y un botón. Al pulsar, mostrar la suma de ambos valores.

Ejercicio 7: Botón de ocultar / mostrar

Archivo: ejercicio7.html
Un párrafo visible y un botón. Permitir ocultar y mostrar el contenido alternando el texto del botón.

Ejercicio 8: Cambiar imagen al hacer clic

Archivo: ejercicio8.html
Una imagen que cambia a otra diferente al hacer clic en un botón.

Ejercicio 9: Lista de tareas sencilla

Archivo: ejercicio9.html
Añadir elementos a una lista (ul) a partir del texto ingresado en un input.

Ejercicio 10: Semáforo interactivo

Archivo: ejercicio10.html
Simular un semáforo con tres estados (rojo, naranja, verde) que cambian al pulsar un botón, mostrando el mensaje correspondiente.
